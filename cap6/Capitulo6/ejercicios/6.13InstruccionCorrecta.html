Ejercicios de autoevaluación

Dada y = ax³ + 7, ¿Cuáles de las siguientes instrucciones de JavasScript son correctas para esta ecuación?

a) y = a * x * x + 7;

Incorrecta

b) y = a * x * x * (x + 7);

Incorrecta

c) y = (a * x) * x * (x + 7);

Correcta

d) y = (a * x) * x * x +1;

Incorrecta

e) y = a * (x * x * x) + 7;

Correcta

f) y = a * x * (x * x + 7);

Incorrecta
